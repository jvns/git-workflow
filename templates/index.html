Visualize your Git!

  • By Julia Evans

Hello! You can use this tool to visualize how you use Git!

First, run this command. If you have bash aliases, try replacing history with this script. 

history | awk '$2 == "git" {print $1 " " $3}' > history.txt

Next, paste the results! You'll get a a graph of which commands you use after other commands. For example, if you always push after committing, there will be a big arrow from 'commit' to 'push'.
